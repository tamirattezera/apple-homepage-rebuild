# 🍎 Apple Homepage Rebuild

## Project Overview

This project is a **front-end rebuild of the current Apple.com homepage** using **HTML and CSS**.  
The goal is to replicate the **modern, clean, and responsive design** while maintaining a professional and scalable folder structure.

## Project Goals

- Practice Semantic HTML
- Master CSS Layout (Flexbox & Grid)
- Improve UI Architecture Skills
- Learn Responsive Design
- Build Component-Based UI
- Develop Professional Git Workflow

## What I'm Learning

- Semantic HTML Structure
- CSS Grid & Flexbox
- Responsive Design
- UI Architecture
- Component-Based Design
- Professional Git Workflow

## Features

- Fully responsive homepage layout
- Rebuilt **Apple navigation bar** with icons (Search, Bag, Menu)
- Hero sections with high-quality product images
- Structured **assets folder** for logos, icons, and images
- Semantic HTML5 elements for accessibility
- Modular CSS for easy maintenance and updates
